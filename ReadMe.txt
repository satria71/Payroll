Cara Instalasi

*Aplikasi yang di butuhkan:

1. Xampp version 3.2.2
2. Php version 7.2.10

Cara instalasi
1. Install Xampp seperti biasa
2. Buka Xampp nyalakan Apache dan MySql
3. Kemudian copas folder aplikasi payroll kedalam folder xampp/htdocs
4. Import database payroll dengan cara masuk phpmyadmin pada browser ketik localhost/phpmyadmin
5. Buat database baru bernama payroll
6. Pilih import kemudian browse lalu pergi xampp/htdocs/payroll pilih file bernama payroll.sql kemudian ok
7. Tunggu proses import hingga selesai
8. Untuk mengakses website buka browser kemudian ketik url localhost/payroll
9. Login dengan username admin dan password admin
